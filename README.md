🚀 FORBODY ACADEMIA: Landing Page de Franquia
Este é o repositório da Landing Page otimizada para captação de leads de franquia para a FORBODY ACADEMIA. O objetivo é apresentar o modelo de negócio de forma clara, gerar prova social e capturar propostas de investimento qualificadas.

✨ Recursos e Funcionalidades
O site é uma Single Page Application (SPA) estática otimizada para performance e conversão, utilizando HTML, CSS e JavaScript puro.

Design Responsivo (Mobile First): Layout adaptável para smartphones, tablets e desktops.

Tema Claro/Escuro: Suporte a $@media (prefers-color-scheme)$ para alternar automaticamente entre os temas.

Menu Fixo (Sticky Header): A navegação permanece visível no topo da tela.

Seção de FAQ Interativa: Utiliza o componente "Acordeão" (Accordion) com JavaScript para expandir e recolher respostas.

Formulário de Contato Robusto: Utiliza Formspree para backend de envio de e-mails e possui validação de campos no lado do cliente (script.js) para garantir a qualidade dos leads (checa Nome/Sobrenome, Email, Telefone e Seleção de Investimento).

Otimização de Conversão (CTA): Chamadas para Ação (CTAs) claras e com destaque na paleta de cores Bordô.

Tecnologia,Descrição:
HTML5,Estrutura semântica e acessível.
CSS3,Estilização com Variáveis CSS para fácil manutenção do tema.
JavaScript (ES6),"Funcionalidades de Menu, FAQ e Validação de Formulário."
Formspree,Serviço de backend para recebimento de dados do formulário (endpoint: https://formspree.io/f/mjknoabd).

Arquivo,Conteúdo:
index.html,"Estrutura principal, conteúdo das seções e formulário."
styles.css,"Definição de variáveis de cor, layout, responsividade e estilos do FAQ."
script.js,"Lógica de interação: menu mobile, FAQ e validação de formulário."
logo.png,Arquivo de imagem do logotipo da FORBODY (Nome ajustado para convenção).
fundo-academia.jpg,(Apenas arquivo de referência. Não está sendo usado no index.html final).
